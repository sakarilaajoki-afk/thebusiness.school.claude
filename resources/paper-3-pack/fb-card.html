FB / LinkedIn cover image — how to use

  1. Maximize this browser window — the card below is exactly 1200×630 px (Facebook + LinkedIn recommended size).
  2. Press Win + Shift + S (Windows) or Cmd + Shift + 4 (Mac) — opens the snipping tool.
  3. Drag a selection around the dark card below (just the card, not this instruction block).
  4. Copy the screenshot (auto-copied on Windows) or save it. Paste/upload to your Facebook post or LinkedIn post.
  5. The card already has the title, subtitle, key stats, and URL — no need to add text in FB.
