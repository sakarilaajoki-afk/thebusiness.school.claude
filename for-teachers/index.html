Before you run your first session

A short checklist:
  • Open a live session on a laptop connected to a projector.
  • Generate a PIN from the teacher dashboard.
  • Have a class set of devices available (Chromebooks, laptops, tablets, or phones).
  • Pre-assign teams of 3–5 if possible — saves time at the start of the lesson.
  • If your school firewall blocks unknown domains, ask IT to allow thebusiness.school.
